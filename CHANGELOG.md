# 更新日志

## [2.0.0] - 2026-04-28

### 核心功能 (v1.1.0 基础)
- 衣橱管理：衣物录入、编辑、删除、闲置/唤醒状态切换
- 穿搭管理：创建穿搭组合、穿搭日志记录
- 首页：Bento Grid 风格布局，快速导航与概览
- 个人中心：设置与偏好管理
- 主题系统：多配色方案（冰川蓝、翡翠绿等），深色/浅色模式支持
- 数据库：基于 sqflite 的本地数据持久化（衣物、穿搭、地点、操作日志）
- UI 组件：GlassCard、GlassNavBar 等玻璃拟态风格组件

### 新增功能 (v1.2.0 更新)
- **天气模块重构**：
  - 改用 Open-Meteo API（免费、无需 Key、全球支持）
  - 默认基于定位获取天气，支持手动切换城市
  - 预报扩展至 7 天
  - 天气卡片支持定位/城市模式一键切换
- **相机增强**：
  - 新增放大功能：双指捏合缩放、双击切换 1x/2x
  - 预设档位按钮（0.5x/1x/2x/5x）与滑动条微调
  - 支持直接调用系统相机拍照
- **地点管理优化**：支持地址显示与编辑，定位获取更稳定

### 修复与优化
- 修复缓存清理逻辑，确保数据正确清除
- 修复地点管理获取定位失败问题（增加 `getLastKnownPosition` 兜底）
- 修复地点管理页地址文本溢出问题
- 修复桌面端不支持定位导致天气获取失败的问题
- 修复墨迹天气反爬虫导致获取失败的问题
- 主题模式默认跟随系统 (`ThemeMode.system`)
- 优化相机缩放体验（仿小米 Leica 相机策略）
